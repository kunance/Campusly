the $user profile
-----------------

given that:

    each time we write a bid we also write a record:

    key                                 value
    ---                                 ---------
    bids/security/<ownerId>/<tenantId>: <bidId>
    bids/security/simplelogin:38/simplelogin:29   : <bidId>

    only writable by <tenantId> and only if <bidId> is by <tenantId>.

 can be .read

    * from the user itself: auth.uid === $user.

    * from an owner that has receives a bid: root.child('bids/security/'+auth.uid+'/'+$user).exists()

    * from a tenant: root.child('bids/relations/'+$user+'/'+auth.uid).exists()

           . that has an accepted bid: root.child('bids/all/'+root.child('bids/relations/'+$user+'/'+auth.uid).val()+'/accepted').exists())
   
 can be .write
    * only by the user itself: 
        . and only updated: newData.hasChildren()
        . or deleted: !newData.exists()
